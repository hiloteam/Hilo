WebAudio

模块 hilo/media/WebAudio 依赖 hilo/core/Class, hilo/util/util, hilo/event/EventMixin 源码 WebAudio.js
WebAudio声音播放模块。它具有更好的声音播放和控制能力，适合在iOS6+平台使用。 兼容情况：iOS6+、Chrome33+、Firefox28+支持，但Android浏览器均不支持。
