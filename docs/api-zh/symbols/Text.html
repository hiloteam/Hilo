Text

继承 Text View 模块 hilo/view/Text 依赖 hilo/core/Class, hilo/core/Hilo, hilo/view/View, hilo/view/CacheMixin 源码 Text.js
Text类提供简单的文字显示功能。复杂的文本功能可以使用DOMElement。
