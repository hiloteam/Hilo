HTMLAudio

模块 hilo/media/HTMLAudio 依赖 hilo/core/Hilo, hilo/core/Class, hilo/event/EventMixin 源码 HTMLAudio.js
HTMLAudio声音播放模块。此模块使用HTMLAudioElement播放音频。 使用限制：iOS平台需用户事件触发才能播放，很多Android浏览器仅能同时播放一个音频。
