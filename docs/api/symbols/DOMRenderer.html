DOMRenderer

继承 DOMRenderer Renderer 模块 hilo/renderer/DOMRenderer 依赖 hilo/core/Class, hilo/core/Hilo, hilo/renderer/Renderer, hilo/view/Drawable 源码 DOMRenderer.js
DOM+CSS3渲染器。将可视对象以DOM元素方式渲染出来。舞台Stage会根据参数canvas选择不同的渲染器，开发者无需直接使用此类。

属性概览

显示继承属性
属性 定义于
canvas:Object
渲染器对应的画布。它可能是一个普通的DOM元素，比如div，也可以是一个canvas画布元素。只读属性。
Renderer
renderType:String
渲染方式。只读属性。
Renderer
stage:Object
渲染器对应的舞台。只读属性。
Renderer
